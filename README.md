# Bubbles Uncertainty

This repository provides the implementation of the proposed method for uncertainty-aware bubble and lesion segmentation.

This is an **anonymous repository** created for peer review purposes.

---

## Repository Structure

- `Encoder/`, `Decoder/`  
  Network architecture components.

- `Case_level_evaluation/`  
  Scripts for case-level evaluation and result aggregation.

- `threshold_optimization_coverage_test/`  
  Threshold optimization and coverage experiments.

- `train_*.py`  
  Training scripts for different model settings.

- `inference_*.py`  
  Inference scripts used to generate segmentation results.

---

## Dataset

Due to privacy and data usage restrictions, the datasets and annotations are **not included** in this repository.

Please refer to the paper for details on dataset preparation and experimental settings.

---

## Pretrained Models

Pretrained weights are not included in this anonymous repository.
Instructions for obtaining or training models are described in the paper.
